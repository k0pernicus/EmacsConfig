EmacsConfig
===========

My emacs config (for MacOS) - Emacs 24.2

Pour Ocaml, veuillez vous assurer de l'avoir déjà installé.

Téléchargez la version 24.2 d'emacs ici: andre.lovichi.free.fr/teaching/ea/2013-2014/install/mac/Emacs-24.2-universal-10.6.8.dmg

Déplacez tous les fichiers .el dans /Applications/Emacs.app/Contents/Resources/site-lisp

Renommez le fichier emacs en .emacs: mv emacs ~/.emacs

Profitez de votre nouvelle configuration emacs!
